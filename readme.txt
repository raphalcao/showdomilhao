Documentação do sistema de perguntas e respostas com base no programa show do milhão.

Este documento tem por finalidade instruir a utilização do programa em questão.

Para a funcionalidade do programa é necessário executar a partir do arquivo app.py.

O arquivo app.py contém a lógica principal do sistema, onde executa todos os processos da dinâmica do programa.

O arquivo module.py está localizado todas as perguntas junto com as respectivas respostas.

O arquivo funcao.py contém as funções de validação de alternativa escolhida e a função do cálculo do valor a receber.


A dinâmica do programa é executada da seguinte maneira:

- Quando o participante acerta determinada questão, o sistema avança para a próxima questão;
- Quando o participante erra a questão, o sistema contabiliza os acertos e faz o cálculo do valor a receber pelo participante.

O sistema é interativo e de fácil entendimento.
